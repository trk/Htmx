# Htmx Module for ProcessWire CMS/CMF



## Requirements

* ProcessWire `3.0.173` or newer
* PHP `7.0` or newer

## Installation

Install the module from the [modules directory](https://modules.processwire.com/modules/htmx/):

Via `Composer`:

```
composer require trk/htmx
```

Via `git clone`:

```
cd your-processwire-project-folder/
cd site/modules/
git clone https://github.com/trk/Htmx.git
```
